# Тестовое задание: Laravel

## Что реализовано
- Миграции и связи моделей: User, Position, Category, Car, Driver
- Сидеры с пользователями, машинами, категориями и водителями
- API маршрут `/api/cars/available`:
  - фильтрует машины по категории пользователя
  - фильтрует по имени машины
  - возвращает JSON с информацией о категории и водителе

## Запуск проекта
1. Клонировать репозиторий
2. `composer install`
3. Скопировать `.env.example` в `.env` и при необходимости настроить базу данных
4. Создать базу данных
5. `php artisan migrate --seed`
6. `php artisan serve`
